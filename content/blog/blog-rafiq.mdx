# Mengenal Next.js: Framework React untuk Produksi

Next.js adalah framework React yang sangat populer untuk membangun aplikasi web modern. Next.js memudahkan kamu membuat aplikasi yang cepat, SEO-friendly, dan mudah diskalakan.

## Kenapa pilih Next.js?

- **Rendering Hybrid:** Mendukung Static Site Generation (SSG) dan Server-Side Rendering (SSR).
- **Routing Otomatis:** Cukup buat file di folder `pages`, otomatis jadi route.
- **API Routes:** Bisa buat backend ringan langsung di aplikasi Next.js.
- **Optimasi Gambar:** Built-in Image Optimization.
- **Dukungan TypeScript:** Plug and play untuk TypeScript.
